### 0. Quickstart
1. Obtenez une clé [OpenAI API](https://platform.openai.com/account/api-keys).
2. Vérifiez la configuration de votre environnement Python (3.10+).

### 1. Bootstrapping
```bash
git clone https://github.com/votre-organisation/Sovereign-Omega-AI.git
cd Sovereign_Omega_Core/AutoGPT-0.4.7
pip install -r requirements.txt
cp .env.template .env
```
*Configurez le fichier `.env` pour activer les modules OMEGA spécifiques.*

### 2. Ignition
```bash
python -m autogpt
```
